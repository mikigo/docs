# 麻辣小龙虾

import { Steps } from '@theme';

<Steps>
### 第 1 步

准备材料：

- 小龙虾 3 斤，清洗、去头、开背、去虾线
- 名扬牛油特辣火锅底料 1 袋，用 1/3
- 辅料：洋葱、大蒜、姜、花椒、大葱、二荆条、干辣椒、小米椒、泡海椒
- 香料：香叶、桂皮、八角、花椒
- 配料：小葱、香菜、土豆
- 其他视情况可以加，比如：酸菜


### 第 2 步

- 锅烧热，加宽油
- 油温 7 成热，倒入小龙虾
- 炸至金黄，捞出备用

### 第 3 步

- 锅留底油，加热到 8 成热
- 倒入辅料、香料、火锅底料，使劲翻炒（注意：火锅底料不能大火炒太久）
- 倒入小龙虾，继续翻炒
- 加水（可以多加一点，使劲煮，消灭细菌病毒）
- 调味：盐巴、鸡精、味精，凭感觉加


### 第 4 步

- 起锅前 10 分钟，倒入土豆（老婆喜欢吃粉土豆，所以煮久一点）
- 起锅前 1 分钟，倒入小葱、香菜各一半
- 起锅后把剩下的小葱、香菜洒在上面

</Steps>
