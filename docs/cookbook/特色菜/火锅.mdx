# 火锅（家庭版）

import { Steps } from '@theme';

<Steps>
### 第 1 步

准备材料：

- 名扬牛油特辣火锅底料 1 袋
- 肉：肥牛、羊肉卷、麻辣牛肉，鸡脚、鸭脚、火腿肠
- 鱼：耗儿鱼、黄辣丁（如果吃鱼火锅，就准备鲢鱼就行）
- 经典套餐：毛肚、千层、黄喉、鸭肠
- 素菜：土豆长丝、笋片、茼蒿、豌豆尖
- 小吃：小酥肉、红糖糍粑
- 辅料：大蒜、小葱、香菜、香油、耗油

### 第 2 步

- 把所有菜洗干净，切好装盘
- 所有辅料备好，装碗，自己根据需要用
- 火锅底料加水，煮开（水可以多加一点点，煮开之后小火慢煮）
- 把一些经煮的先放进去煮，其他的自己吃的时候在涮

</Steps>
